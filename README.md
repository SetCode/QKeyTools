# QKeyTools

1、此代码是基于Qt平台下的软件盘实现，适合用于Qt4和Qt5.8以下的Qt版本；

2、主要使用的场景：
  2.1、嵌入式平台；
  2.2、桌面软键盘；
  2.3、自定义输入软键盘；
 
3、实现的功能：
  3.1、能够自定义键盘的大小；
  3.2、能够指定键盘的各种样式；
  3.3、能够指定键盘的显示位置（提供按钮正下方、屏幕中间、屏幕底部）
  3.4、针对嵌入式平台的键盘显示方式；
